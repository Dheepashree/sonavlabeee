SPEED CONTROL OF DC SHUNT MOTOR

FORMULA USED

1. Radius of brake drum r = circumference / (2π) in metre

2. Torque T = 9.81 *F*r in N-m

3. Input Power Pin =Vin * Iin in Watts

4. Output power Pout = 2πNT / 60 in watts

5. Percentage Efficiency η = (Pout / Pin ) x 100 %

Where,

Vin=Input voltage in Volts

Iin= Input current in Amperes

N= Speed of the motor in rpm

T= Torque in Nm.
