LOAD CHARACTERISTICS OF DC SHUNT MOTOR

MODEL CALCULATION 

 Reading No. 2 
				SPRING BALANCE READING (F) CALCULATION : 
					F1=0 Kg 
					F2=1.5 Kg 
					F=F2-F1 = 1.5 Kg 
				TORQUE (T) CALCULATION :
					F = 1.5 Kg
					Radius (r) = 0.1 m
					T = 9.81 X F X r
					  = 9.81 X 1.5 X 0.1
					T = 1.4719 Nm.
				INPUT POWER (Pin) CALCULATION : 
					Input Voltage (Vin) = 224 V
					Input Current (Iin) = 2.6 A
					Pin = Vin X Iin
						= 224 X 2.6
					Pin	= 582.4 W
				OUTPUT POWER (Pout) CALCULATION :
					Speed (N) = 1485 rpm
					Torque (T) = 1.4715 Nm
					Pout = (2πNT)/60
						 = (2 X 3.14 X 1485 X 1.4715)/60
					Pout = 228.7 W
				EFFICIENCY (η) CALCULATION : 
					Pin = 582.4 W
					Pout = 228.7 W
					Efficiency = (Pout/Pin) X 100
							   = (228.7/582.4) X 100
					         η = 39.25 %
