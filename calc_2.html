MODEL CALCULATION 

 Reading No. 1 
				SPRING BALANCE READING (F) CALCULATION : 
					F1=2 Kg 
					F2=19 Kg 
					F=F2-F1 = 17 Kg 
				TORQUE (T) CALCULATION :
					F = 17 Kg
					Radius (r) = 0.1 m
					T = 9.81 X F X r
					  = 9.81 X 17 X 0.1
					T = 16.67 Nm.
				INPUT POWER (Pin) CALCULATION : 
					Input Voltage (Vin) = 220 V
					Input Current (Iin) = 12 A
					Pin = Vin X Iin
						= 220 X 12
					Pin	= 2640 W
				OUTPUT POWER (Pout) CALCULATION :
					Speed (N) = 1134 rpm
					Torque (T) = 16.67 Nm
					Pout = (2πNT)/60
						 = (2 X 3.14 X 1134 X 16.67)/60
					Pout = 1978 W
				EFFICIENCY (η) CALCULATION : 
					Pin = 2640 W
					Pout = 1978 W
					Efficiency = (Pout/Pin) X 100
							   = (1978/2640) X 100
					         η = 74.9 %
